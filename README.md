<h1>Overview</h1>

A digital implementation of the classic board game Pete's Pike, built in Java with a graphical user interface. 
This project codifies the full game experience, allowing a single player to engage with the game digitally.

<h1>About the Game</h1>

Pete's Pike is a strategic board game where the player navigates challenges across a mountain-themed board, 
making decisions that determine their path to the summit. This implementation faithfully recreates the rules 
and structure of the original game in a fully interactive Java application.

<h1>Features</h1>

Single Player Experience — Play through the full game at your own pace<br>
Graphical User Interface — Clean, interactive GUI built with Java for an intuitive experience
